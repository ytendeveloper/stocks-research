# Stocks Research Hub

A curated research workspace for tracking and analyzing stocks across multiple categories.

## Structure

- **categories/** — One markdown file per category listing all stocks in that group
- **stocks/** — One detailed HTML file per stock with full research data
- **dashboard.html** — Interactive dashboard with summary view of all categories and stocks

## How to Use

1. Open `dashboard.html` in a browser for an interactive overview
2. Browse `categories/` for curated stock lists by theme
3. Open any file in `stocks/` for deep-dive research on a single ticker
